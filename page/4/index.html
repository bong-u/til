🏫 학과 공부
네트워크 보안 개요 네트워크 보안의 요구사항 기밀성(Confidentiality) 정보를 권한이 없는 개인에게 노출되지 않도록 함 무결성(Integrity) 정보와 프로그램은 인가된 방식으로만 변경되도록 함 가용성(Availability) 정보 자산에 대해 적절한 시간에 접근 가능하도록 함 보안 공격의 종류 소극적 공격(Passive Attack) 정보를 도청하거나 감시하는 공격 방어방법 : 암호화 적극적 공격(Active Attack) 정보를 변조하거나 삭제하는 공격 방어방법 : 메시지 인증 보안 서비스 인증(Authentication) 사용자의 신원을 확인하는 과정
