React - useEffect 사용법

🏷️ react

수정일 : 2024-11-15

  1. component가 rendering 될 때 실행
    1useEffect(()=> {
2    ...
3});
  2. component가 mount 될 때 (처음 rendering 될 때) 실행
    1useEffect(()=> {
2    ...
3}, []);
  3. component가 update 될 때 (props, states가 바뀔 때) 실행
    1useEffect(()=> {
2    ...
3}, [{ props or states }]);
  4. component가 unmount 될 때 또는 update되기 직전에 실행
    1useEffect(()=> {
2    return () => {
3        ...
4    };
5}, []);
